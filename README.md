Интернет‑приложение для составления жалобы на парковку самокатов. Автоматически подцепляет адрес и время из фото, нужно только ввести номер самоката


Сформированный текст можно отправить через функцию телефона “Поделиться”: открываете через почтовый клиент, где нужно вставить скопированный адрес почты. Приложение можно установить на телефон как PWA (Добавить на главный экран).
